# Auto Loan Securitisation Risk Assessment Dashboard

## Project Overview

This project focuses on building an **Auto Loan Securitisation Analytics Dashboard** using **Microsoft Power BI**. The objective is to analyze securitisation risk, portfolio performance, delinquency trends, dynamic losses, vintage analysis, and stress testing metrics using assumed banking/ERP datasets.

## Tools & Technologies

* Microsoft Power BI
* DAX (Data Analysis Expressions)
* Power Query
* Excel / CSV Datasets
* Data Modeling

## Datasets Used

* **FactLoanPerformance**
* **FactDPDSnapshot**
* **FactDynamicLoss**
* **FactStaticPool**
* **DimDate**

## Key Dashboards

### 1. Pool Summary / Performance Overview

* MTD Analysis
* QTD Analysis
* YTD Analysis

### 3. DPD & Risk Monitoring Dashboard

### 4. Vintage / Static Pool Analysis

* Cumulative Loss Curves
* Static Pool Performance

### 5. Dynamic Loss Dashboard

* CPR / CDR Metrics

### 6. Waterfall & Tranche Allocation

* Cashflow Analysis
* Waterfall Visualization
* Tranche Performance Concepts

### 7. IFRS-9 & Stress Testing

* Stage 1 / 2 / 3 Framework
* Scenario Analysis
* What-If Parameters
